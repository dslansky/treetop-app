# Treetop ABA — Family App

Interactive prototype for the Treetop ABA client-facing mobile app.

**Live preview:** [dslansky.github.io/treetop-app](https://dslansky.github.io/treetop-app)

## Prototype screens

- Login / Sign Up / Pending Approval
- Home Dashboard (child switcher, today's sessions, messages, quick actions)
- Schedule (day picker, session timeline)
- Session Detail / Cancel / Reschedule
- Chat (continuous thread, text + photos, document sharing)
- Documents (shared files + parent uploads)
- Care Team (staff profiles)
- Settings (account, notifications, calendar sync)

## Status

Planning & prototype phase. No production code yet.
